## 常见问题排查

### Q1: 登录后还是看不到"后台管理"入口

**排查步骤**：
1. 检查是否真的登录成功（右上角是否显示用户昵称）
2. 刷新页面
3. 清除浏览器缓存后重新登录
4. 检查浏览器控制台是否有错误

### Q2: 点击"后台管理"没有反应

**排查步骤**：
1. 检查浏览器控制台是否有错误
2. 检查URL是否变化为 `#/admin`
3. 尝试直接在地址栏输入 `#/admin`

### Q3: 配置保存后刷新页面丢失

**排查步骤**：
1. 检查浏览器是否允许localStorage
2. 打开浏览器开发者工具 → Application → Local Storage
3. 查看是否有 `admin_api_config` 和 `admin_system_config` 键
4. 检查是否在隐私模式下浏览

## 技术说明

### 修复的技术问题

**问题1：导航菜单不更新**
```typescript
// ❌ 错误：直接修改常量数组
const navItems = [...];
if (currentUser) {
  navItems.push(...); // 这不会触发重新渲染
}

// ✅ 正确：使用useMemo动态生成
const navItems = useMemo(() => {
  const items = [...];
  if (currentUser) {
    items.push(...);
  }
  return items;
}, [currentUser, t]);
```

**问题2：useEffect无限循环**
```typescript
// ❌ 错误：依赖项包含函数
useEffect(() => {
  // ...
}, [t, currentUser, navigate, toast]); // navigate和toast会导致循环

// ✅ 正确：只依赖必要的值
useEffect(() => {
  // ...
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, [t]);
```
