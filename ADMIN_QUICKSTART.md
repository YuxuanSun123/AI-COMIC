# 后台管理快速使用指南

## 🚀 快速开始

### 1. 访问后台管理

**步骤**：
1. 打开应用
2. 点击右上角"注册"或"登录"
3. 登录成功后，顶部导航栏会出现"后台管理"入口
4. 点击"后台管理"进入配置页面

**URL**: `#/admin` 或 `/admin`

---

## ⚙️ 配置OpenAI API

### 方式一：使用OpenAI官方API

1. 进入后台管理 → **API配置** 标签
2. 填写配置：
   ```
   OpenAI API Key: sk-your-api-key-here
   API Base URL: https://api.openai.com/v1
   模型名称: gpt-4
   ```
3. 开启"启用AI生成"开关
4. 点击"测试连接"验证
5. 点击"保存配置"

### 方式二：使用Azure OpenAI

1. 进入后台管理 → **API配置** 标签
2. 填写配置：
   ```
   OpenAI API Key: your-azure-api-key
   API Base URL: https://your-resource.openai.azure.com/
   模型名称: your-deployment-name
   ```
3. 开启"启用AI生成"开关
4. 点击"保存配置"

### 方式三：使用本地模拟（默认）

1. 进入后台管理 → **API配置** 标签
2. 关闭"启用AI生成"开关
3. 点击"保存配置"
4. 系统将使用本地模拟生成内容

---

## 📊 数据管理

### 导出数据备份

1. 进入后台管理 → **数据管理** 标签
2. 查看数据统计（用户数、作品数、新闻数、链接数）
3. 点击"导出所有数据"
4. 下载JSON文件：`backup-{时间戳}.json`

### 清空所有数据

1. 进入后台管理 → **数据管理** 标签
2. 点击"清空所有数据"
3. 确认操作（⚠️ 不可逆）
4. 页面自动刷新

---

## 🔧 系统设置

### 修改网站信息

1. 进入后台管理 → **系统设置** 标签
2. 修改：
   - 网站名称
   - 网站描述
   - 每用户最大作品数
3. 点击"保存设置"

### 切换存储模式

**本地存储**（默认）：
- 数据存储在浏览器localStorage
- 无需后端服务器
- 适合演示和原型

**远程API**：
- 数据存储在远程服务器
- 需要配置后端API地址
- 适合生产环境

---

## 🎯 使用场景

### 场景1：开发测试（本地模拟）

```
1. 不配置API Key
2. 关闭"启用AI生成"
3. 使用剧本生成器 → 生成模拟内容
```

### 场景2：真实AI生成

```
1. 配置OpenAI API Key
2. 开启"启用AI生成"
3. 使用剧本生成器 → 调用真实API生成
```

### 场景3：数据迁移

```
1. 旧环境：导出所有数据
2. 新环境：导入数据（手动复制到localStorage）
3. 验证数据完整性
```

---

## ⚠️ 注意事项

### 安全提示

1. **API Key安全**：
   - 存储在浏览器localStorage
   - 仅演示用途
   - 生产环境应使用后端代理

2. **数据备份**：
   - 定期导出数据备份
   - 清除浏览器数据会丢失配置

3. **权限控制**：
   - 当前版本仅检查登录状态
   - 所有登录用户都可访问后台

2. 滚动到底部
3. 点击"后台管理"
4. 所有功能与桌面端一致

---

## 🔍 故障排查

### 问题1：配置了API Key但还是本地模拟

**解决**：
- 检查是否开启"启用AI生成"开关
- 检查是否点击"保存配置"
- 刷新页面重新加载配置

### 问题2：导出的数据文件为空

**解决**：
- 检查localStorage是否有数据
- 检查浏览器是否允许下载文件
- 尝试使用其他浏览器

### 问题3：清空数据后无法恢复

**解决**：
- 清空操作不可逆
- 如有备份文件，手动导入localStorage
- 重新初始化模拟数据
